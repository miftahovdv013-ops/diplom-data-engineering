# Дипломный проект: Инженер данных

## Разработка ETL процессов и хранилища данных для розничных продаж

### Описание проекта
Разработан полный пайплайн обработки данных от CSV-файла до интерактивных дашбордов.

### Технологии
- **PostgreSQL 15** — хранилище данных
- **Python 3.9** — ETL процессы
- **Tableau Public** — визуализация
- **Pandas** — обработка данных
- **Scikit-learn** — ML модели

### Архитектура
- NDS (Normalized Data Store) — 3NF, без дубликатов
- DDS (Dimensional Data Store) — схема звезда
- Data Marts — витрины для Tableau

### Результаты
- 1000 транзакций
- Общая выручка: $318,222
- Средний чек: $323
- 3 дашборда в Tableau
- ML прогноз на неделю: $27,217
